MySQL CRUD Application
This project is a Python-based application that implements basic Create, Read, Update, and Delete (CRUD) operations on a MySQL database. It provides a simple command-line interface to manage user data, including names, ages, and cities.

Features
Insert User Data: Add new user records to the database.
Update User Data: Modify existing user records using their ID.
Display User Data: Retrieve and show all user records in a formatted table.
Delete User Data: Remove user records from the database by ID.
Technologies Used
Python 3.x: The programming language used for the application.
MySQL: The database system used to store user data.
mysql.connector: Python library for MySQL database connections.
tabulate: Library for displaying tabular data in the console.
